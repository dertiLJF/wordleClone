NotWordle


NotWordle is a word-guessing game that is definitely not the same as Wordle.
To play, guess a 5-letter word in 6 tries (or less).

CodeBreaker


CodeBreaker is a number guessing game! Input some numbers and it will give you hints.
An arrow will appear for any incorrect numbers signifying that the number is either (up arrow) higher, or (down arrow) lower!
